# Random-Quote-Generator-HTML-CSS-JS

This **Random Quote Generator** is a simple web application designed to display random quotes each time the user clicks the "Generate Quote" button. It utilizes **HTML**, **CSS**, and **JavaScript** to create a clean and interactive user experience.

### Features:
- Displays a random inspirational or motivational quote with every click.
- Clean and minimalistic design with easy-to-read typography.
- Fully responsive layout, ensuring a smooth experience across devices.
